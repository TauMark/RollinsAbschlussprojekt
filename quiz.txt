 Multiple‑Choice (Single Choice)
1. Was beschreibt Pseudocode am besten?

A) Eine informelle, sprachnahe Beschreibung eines Algorithmus
B) Eine ausführbare Programmiersprache
C) Ein Datenbankmodell
D) Ein UML-Diagramm
Richtige Antwort: A

2. Welche Struktur wird typischerweise für Wiederholungen im Pseudocode genutzt?

A) IF
B) WHILE
C) CLASS
D) IMPORT
Richtige Antwort: B

3. Welche Kontrollstruktur dient zur Entscheidung?

A) LOOP
B) RETURN
C) METHOD
D) IF–ELSE
Richtige Antwort: D

4. Was ist ein Objekt in der OOP?

A) Eine Datei
B) Eine Instanz einer Klasse
C) Eine globale Variable
D) Eine Funktion
Richtige Antwort: B

5. Welche Schleife eignet sich, wenn die Anzahl der Wiederholungen bekannt ist?

A) WHILE
B) REPEAT UNTIL
C) FOR
D) FOREACH
Richtige Antwort: C

6. Welche typische Schreibweise beschreibt eine Methode im Pseudocode?

A) OBJECT Name
B) PACKAGE Name
C) LOOP Name
D) METHOD Name(Parameter)
Richtige Antwort: D

7. Welche Datenstruktur speichert Elemente in Reihenfolge?

A) Boolean
B) Klasse
C) Liste
D) Methode
Richtige Antwort: C

8. Welche Schleife wird mindestens einmal ausgeführt?

A) WHILE
B) FOR
C) REPEAT UNTIL
D) FOREACH
Richtige Antwort: C

9. Welche Aussage beschreibt Vererbung?

A) Eine Klasse übernimmt Eigenschaften einer anderen Klasse
B) Eine Klasse kann nur eine Methode haben
C) Vererbung existiert nicht
D) Eine Klasse kann keine Attribute besitzen
Richtige Antwort: A

10. Was beschreibt Polymorphie?

A) Objekte können unterschiedliche Implementierungen derselben Methode haben
B) Klassen haben keine Attribute
C) Methoden können nicht überschrieben werden
D) Polymorphie existiert nicht
Richtige Antwort: A
